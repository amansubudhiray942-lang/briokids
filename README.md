# 🌟 OT Companion App

A complete Occupational Therapy companion app for children with 54+ activities targeting Autism, ADHD, Sensory Processing, Cerebral Palsy, and Developmental Delays.

## ✨ Features

- 👶 **Child Profile Setup** - Personalize by age, condition, and therapy goals
- 🧩 **54+ OT Activities** - Curated activities for:
  - Sensory Integration
  - Fine Motor Skills
  - Gross Motor Skills
  - Cognitive Development
  - Social Skills
- 📋 **Activity Tracker** - Log sessions with mood feedback and notes
- 📊 **Progress Dashboard** - Track streaks, mood patterns, and milestones
- 💡 **Parenting Tips** - Expert guidance and OT concepts
- 📹 **YouTube Integration** - Instructional videos for each activity
- 📱 **Fully Responsive** - Works on mobile, tablet, and desktop

## 💻 Local Development

### Prerequisites
- Node.js 14+ ([Download here](https://nodejs.org))
- npm (comes with Node.js)

### Installation & Running

```bash
# 1. Clone the repository
git clone https://github.com/amansubudhiray942-lang/briokids.git
cd briokids

# 2. Install dependencies
npm install

# 3. Start development server
npm run dev
```

The app will automatically open at `http://localhost:3000`

### Build for Production

```bash
npm run build
npm run preview
```

## 🌐 Deploy Online (Free Options)

### Option 1: Vercel (Recommended)
1. Go to https://vercel.com/new/clone?repository-url=https://github.com/amansubudhiray942-lang/briokids
2. Click Import

### Option 2: Netlify
1. Go to [netlify.com](https://netlify.com)
2. Click "New site from Git"
3. Select your GitHub repository
4. Deploy automatically

### Option 3: GitHub Pages
```bash
npm run build
# Upload the 'dist' folder to GitHub Pages
```
